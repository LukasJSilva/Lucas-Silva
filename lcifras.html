LCifras - 1° Semestre

Descrição

  • Plataforma de cifras online com biblioteca ampla e atualizada, cobrindo diversos gêneros musicais. Possui interface intuitiva, vídeos tutoriais, dicas técnicas e comunidade interativa, abrangendo uma ampla variedade de gêneros musicais e artistas, desde clássicos do rock até os mais recentes sucessos pop.
  • Tecnologias: HTML, CSS e JavaScript.
  • Contribuição: desenvolvimento completo (estrutura, design e funcionalidades).
  • Estado: projeto acadêmico inicial, criado em 2023 para a disciplina de Desenvolvimento Web I e composição de nota de Projeto Interdisciplinar.
